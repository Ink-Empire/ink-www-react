# Inked In - Next.js Web Application

This is a web version of the Inked In React Native app, built with Next.js.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Browse tattoo artists
- View artist details and portfolios
- Search for artists by name, style, or location

## Technologies Used

- Next.js
- React
- TypeScript
- CSS Modules

## Project Structure

- `/components` - Reusable UI components
- `/data` - JSON data for artists, styles, etc.
- `/models` - TypeScript interfaces
- `/pages` - Next.js pages
- `/public` - Static assets
- `/styles` - Global CSS styles
